EditorsRepo.nodesfortokens — Function

Create a list of CitableNodes from a list of OrthographicTokens and a node URN.

nodesfortokens(tkns, urn)

Node URNs are extended with an additional level of citation for the individual token. This citation tier is made up of sequential numbers for lexical tokens, and token number + a character for other kinds of tokens (1a, 1b, etc.).
